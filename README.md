# tikett

 [![Build Status](https://travis-ci.org/flopes89/tikett.svg?branch=master)](https://travis-ci.org/flopes89/tikett)
 
Offline file tagging software.

## Setup

```
$ npm install --production-only
$ npm start
```

## Developing

```
$ npm install
$ npm run dev-server & npm run dev-client
```
